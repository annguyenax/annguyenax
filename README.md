## About Me

I am a final-year **Information Security** student at **PTIT**, currently building my path toward a **Security Engineering internship**.

My main interests are:

- **SOC / Blue Team:** IDS rules, alert analysis, network monitoring, detection logic
- **Pentest fundamentals:** Web security and basic pwn through CTF practice
- **DevSecOps:** Docker, CI/CD security scanning, secret scanning, secure deployment
- **AppSec:** JWT authentication, API security, OWASP Top 10, secure backend design

I like learning security by doing the full cycle:

```text
Attack  ->  Detect  ->  Analyze  ->  Fix  ->  Document
```
